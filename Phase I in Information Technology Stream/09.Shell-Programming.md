## üß© **1. Introduction to Shell Programming**

### What is a Shell?

* **Shell** is a **command-line interpreter** that provides a user interface to access the services of the operating system.
* It acts as a **bridge between the user and the kernel**.
* Common Linux shells:

  * **sh** (Bourne shell)
  * **bash** (Bourne Again Shell ‚Äì most widely used) (Most popular on Linux   )
  * **csh** (C Shell)
  * **ksh** (Korn shell)
  * **zsh** (Z Shell)

### Why Shell Programming?

* To **automate repetitive tasks**.
* To perform **system administration** efficiently.
* To **combine multiple UNIX commands** in a single file and execute them together.

---

## üßæ **2. Shell Scripting Basics**

### What is a Shell Script?

* A **text file containing a sequence of shell commands**.
* Executed by the shell **line by line**.
* File extension: `.sh`

#### Example of a simple shell script (`hello.sh`):

```sh
#!/bin/bash
echo "Hello, World!"
```

### Steps to Create and Run a Shell Script

1. Create a new file:

   ```bash
   nano myscript.sh
   ```
2. Add the **shebang line** at the top:

   ```bash
   #!/bin/bash
   ```

* **Note:**

* ### **Structure of a Shell Script**

* ### **Shebang (#!)**

* The first line tells the system which interpreter to use.

* Examples:

* `#!/bin/bash` ‚Üí use Bash
* `#!/bin/sh` ‚Üí use default shell

* ### **Comments**

* Use `#` to add comments.

```sh
# This is a comment
```

   (This line tells the system which shell to use to execute the script.)
   
3. Add commands:

   ```bash
   echo "Hello, SEBI Aspirant!"
   ```
4. Make the file executable:

   ```bash
   chmod +x myscript.sh
   ```
5. Run the script:

   ```bash
   ./myscript.sh
   ```

## üßÆ **3. Shell Variables**

### Types of Variables

| Type                       | Description                 | Example                  |
| -------------------------- | --------------------------- | ------------------------ |
| **User-defined Variables** | Created by the user         | `name="Ravi"`            |
| **System Variables**       | Defined by the system/shell | `$HOME`, `$USER`, `$PWD` |

### Syntax

```bash
varname=value
echo $varname
```

### Rules:

* No spaces before or after `=`.
* Variable names are **case-sensitive**.
* To access a variable, use the `$` symbol.

### Examples

```bash
name="SEBI"
echo "Welcome to $name Grade A IT Prep"
```

---

## üí¨ **4. Shell Script Arguments**

Arguments are **values passed to the script** when it is executed.

### Syntax:

```bash
./scriptname arg1 arg2 arg3
```

### Inside the Script:

| Symbol | Meaning             |
| ------ | ------------------- |
| `$0`   | Script name         |
| `$1`   | First argument      |
| `$2`   | Second argument     |
| `$@`   | All arguments       |
| `$#`   | Number of arguments |

### Example:

```bash
#!/bin/bash
echo "Script Name: $0"
echo "First Argument: $1"
echo "Second Argument: $2"
echo "Total Arguments: $#"
```

## üîÄ **5. Conditional Statements (If Statement)**

### Syntax:

```bash
if [ condition ]
then
   commands
elif [ condition ]
then
   commands
else
   commands
fi
```

### Common Operators:

| Type      | Operators                                | Example              |
| --------- | ---------------------------------------- | -------------------- |
| Numeric   | `-eq`, `-ne`, `-gt`, `-lt`, `-ge`, `-le` | `[ $a -gt $b ]`      |
| String    | `=`, `!=`, `-z`, `-n`                    | `[ "$str" = "abc" ]` |
| File Test | `-e`, `-f`, `-d`, `-r`, `-w`, `-x`       | `[ -f file.txt ]`    |

### Example:

```bash
#!/bin/bash
num=10
if [ $num -gt 5 ]
then
   echo "Number is greater than 5"
else
   echo "Number is less or equal to 5"
fi
```

## üîÅ **6. Loops in Shell**

### a) **For Loop**

```bash
for var in 1 2 3 4 5
do
  echo "Value: $var"
done
```

**Range Loop Example:**

```bash
for i in {1..5}
do
  echo "Iteration $i"
done
```

### b) **While Loop**

```bash
count=1
while [ $count -le 5 ]
do
  echo "Count is $count"
  count=$((count + 1))
done
```

### c) **Until Loop**

```bash
count=1
until [ $count -gt 5 ]
do
  echo "Count is $count"
  count=$((count + 1))
done
```

## üîô **7. Return / Exit Status**

Every command in UNIX returns an **exit status**:

* `0` ‚Üí success
* Non-zero ‚Üí failure

You can use `$?` to check the exit status of the last command.

### Example:

```bash
mkdir testdir
if [ $? -eq 0 ]
then
  echo "Directory created successfully."
else
  echo "Failed to create directory."
fi
```

### Using `exit` Command

```bash
exit 0   # normal exit
exit 1   # indicates an error
```

## ‚öôÔ∏è **8. Basic UNIX Commands (Commonly Used in Shell Scripts)**

| Command         | Description                      | Example                |
| --------------- | -------------------------------- | ---------------------- |
| `pwd`           | Prints current directory         | `pwd`                  |
| `ls`            | Lists files/directories          | `ls -l`                |
| `cd`            | Changes directory                | `cd /home/user`        |
| `mkdir`         | Creates a new directory          | `mkdir folder`         |
| `rmdir`         | Removes empty directory          | `rmdir folder`         |
| `rm`            | Removes files/directories        | `rm file.txt`          |
| `cp`            | Copies files/directories         | `cp a.txt b.txt`       |
| `mv`            | Moves/renames files              | `mv old new`           |
| `cat`           | Displays file contents           | `cat file.txt`         |
| `grep`          | Searches text in files           | `grep "text" file.txt` |
| `echo`          | Displays text                    | `echo "Hello"`         |
| `date`          | Shows system date/time           | `date`                 |
| `chmod`         | Changes file permissions         | `chmod +x file.sh`     |
| `head` / `tail` | Shows first/last lines of a file | `head -n 5 file.txt`   |
| `wc`            | Counts lines, words, characters  | `wc file.txt`          |

### Example 1: Check if a number is even or odd

```bash

  echo "Even Number"
else
  echo "Odd Number"
fi
```

### Example 2: Print 1 to 10 using loop

```bash

  echo $i

```

### Example 3: Check if file exists

```bash

if [ -f "data.txt" ]
then
  echo "File exists."
else
  echo "File not found."
fi
```
