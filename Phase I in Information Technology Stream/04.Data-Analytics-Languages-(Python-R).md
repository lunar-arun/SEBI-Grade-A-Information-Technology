# ðŸ”¶ **2. Regular Expressions (Regex)**

Regex is a pattern-matching technique used for **searching, validating, and manipulating text**.

## **Common Patterns**

* `.` â†’ Matches any character
* `^` â†’ Start of string
* `$` â†’ End of string
* `\d` â†’ Digit
* `\w` â†’ Alphanumeric
* `\s` â†’ Whitespace
* `*` â†’ 0 or more occurrences
* `+` â†’ 1 or more occurrences
* `?` â†’ Optional (0 or 1)
* `{n}` â†’ Exactly n repetitions
* `{n,m}` â†’ Between n and m repetitions
* `[]` â†’ Character class
* `|` â†’ OR
* `()` â†’ Grouping

## **Examples**

* Validate email:
  `^[\w\.-]+@[\w\.-]+\.\w{2,4}$`
* Extract digits:
  `\d+`
* Find words starting with capital letter:
  `[A-Z][a-z]+`

## **Python Usage**

```python
import re
re.findall(r"\d+", "SEBI2025Exam")   # ['2025']
```

# ðŸ”¶ **3. Slicing**

Used to extract sub-parts of **strings**, **lists**, **tuples**, **NumPy arrays**, **Pandas dataframes**.

General format:

```
object[start : stop : step]
```

Examples:

### **String slicing**

```python
s = "SEBIGRADEA"
s[0:4]   # 'SEBI'
s[-3:]   # 'DEA'
```

### **List slicing**

```python
lst = [10,20,30,40,50]
lst[1:4]   # [20,30,40]
lst[::-1]  # reverse list
```
