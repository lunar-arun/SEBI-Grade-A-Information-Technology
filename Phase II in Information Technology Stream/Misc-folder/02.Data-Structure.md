### **Easy (1–10)**

1. Which of the following correctly declares an array of 10 integers in C?
   a) `int arr[10];` ✅
   b) `int arr();`
   c) `array arr[10];`
   d) `int arr{10};`

2. What is the index of the first element in an array?
   a) -1
   b) 0 ✅
   c) 1
   d) Depends on the language

3. What is the time complexity to access an element at a given index in an array?
   a) O(n)
   b) O(log n)
   c) O(1) ✅
   d) O(n^2)

4. Which of the following is NOT a valid way to initialize an array in C?
   a) `int arr[3] = {1,2,3};`
   b) `int arr[] = {1,2,3};`
   c) `int arr[3] = {1,2};`
   d) `int arr[3] = (1,2,3);` ✅

5. What is the index of the last element in an array of size `n`?
   a) n
   b) n-1 ✅
   c) 1
   d) 0

6. Which of the following operations is easiest to perform on arrays?
   a) Insertion at beginning
   b) Deletion in middle
   c) Random access ✅
   d) Insertion in middle

7. If an array has 20 elements, what is the index of the 10th element?
   a) 9 ✅
   b) 10
   c) 11
   d) 20

8. Which of the following is true about arrays?
   a) Arrays can store elements of different types
   b) Arrays have dynamic size in C ✅ (False) → Correct: Arrays have fixed size in C
   c) Arrays allow O(1) insertion at beginning
   d) Arrays are always 2D

9. What will be the output of the following C code?

   ```c
   int arr[5] = {1,2,3,4,5};
   printf("%d", arr[2]);
   ```

   a) 1
   b) 2
   c) 3 ✅
   d) 4

10. Which of these is used to store multiple values of the same type?
    a) Linked List
    b) Array ✅
    c) Stack
    d) Queue
